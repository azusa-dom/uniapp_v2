uniapp · 为学生与家长打造的智能校园助手

基于 SwiftUI 的多角色校园管理体验，融合学习任务、日程提醒、数据可视化与 AI 助手能力。此页面提供一站式概览与核心代码片段，便于在 GitHub Pages / Vercel 等平台快速展示。

项目亮点

学生与家长双端体验

学生端聚焦课程与待办，家长端掌握数据共享与学习进度，所有视图在 SwiftUI 中共享 Design System。

可复用组件库

通过 `CommonComponents.swift` 构建统一卡片、Glassmorphism 风格与可重用空状态视图，方便扩展新模块。

智能任务与活动助手

集成 AI 助手、日程热力图、作业进度等模块，让学业安排更直观。

易于部署展示

纯静态页面 → 可直接托管到 GitHub Pages / Vercel / Netlify，亦可嵌入 FlutterFlow、Framer 的 HTML Section。

如何运行原生 App

本地体验

使用 Xcode 15+ 打开 `uniapp.xcodeproj`，选择模拟器或真机即可运行。建议开启 `AppState` 和 `LocalizationService` 的测试数据。

对外展示

通过 TestFlight 发布或在网页中嵌入演示视频、截图；当前页面的代码片段可作为技术亮点演示。

继续扩展

可在 `assets/` 中补充产品截图，也可以新增更多 Swift 文件到 `code/` 目录，自动被网页读取并高亮。
